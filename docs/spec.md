# TermsOfUs spec

## Goal
Help people clarify relationship expectations and boundaries using a structured grid.

## Core concepts
A user can mark each item as:
- We have / already present
- I want
- I do not want
- Not relevant / unanswered

## V1
- Create a local relationship profile
- Fill a checklist/grid
- Save locally in the browser
- Export/share as JSON or link later
- Compare two profiles later, not in first implementation

## Constraints
- Multilingual from the start: English and French
- Privacy-first
- Local-first by default
- Simple UI
