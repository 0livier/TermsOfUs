
# Ticket 006 - Mobile list view

## Goal

Create the mobile-first structured list experience.

## Read first

- AGENTS.md

- docs/product/v1-summary.md

- docs/product/spec-v1.md sections 7.3, 9.3, 9.4, 10.4, 14.3, 20

- docs/architecture/decisions.md

## Scope

Implement:

- category cards or accordions

- item chips inside categories

- sticky or convenient palette placement

- responsive mobile layout

## Out of scope

- Desktop SVG

- search and filtering

- comparison mode

## Acceptance criteria

- Mobile layout is usable without pinch zoom.

- Touch targets are comfortable.

- Category grouping is clear.

- Mobile view uses the shared selection state.

- Selection updates summary and URL.
